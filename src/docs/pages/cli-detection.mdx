# Detection Methods

AMVerge CLI supports three detection methods. Choose based on your accuracy
and speed needs.

## Keyframe Detection

**Default method.** Fast, no extra dependencies.

Extracts I-frame timestamps from the video using PyAV packet demux. Splits at
keyframe boundaries. Scenes not aligned to keyframes use lossless stream copy.
Non-aligned scenes get smartcut (encode head + copy tail).

```bash
amverge detect episode.mp4 --method keyframe
```

- Speed: fast (seconds for a 24-minute episode)
- Accuracy: good for most anime and editing
- Requires: nothing extra

## TransNetV2 ML Detection

**Deep learning scene boundary detection.** Best accuracy.

Runs a pre-trained CNN on 48x27 RGB frames. Auto-detects GPU (CUDA/MPS),
falls back to CPU. First run downloads model weights (~40MB), cached locally.

```bash
pip install amverge[ml]
amverge detect episode.mp4 --method transnetv2
```

- Speed: medium (GPU: ~2x real-time, CPU: ~0.5x)
- Accuracy: best - catches dissolves, fades, and complex transitions
- Requires: `pip install amverge[ml]`

## Edge Detection

**Canny edges + cosine similarity.** Good for difficult encodes.

Decodes frames and compares Canny edge maps frame-by-frame. Cosine similarity
threshold determines cut points. More accurate than keyframe, slower than
TransNetV2.

```bash
pip install amverge[edge]
amverge detect episode.mp4 --method edge
```

- Speed: slow (CPU only, ~0.2x real-time)
- Accuracy: excellent for hard cuts and fast action
- Requires: `pip install amverge[edge]`

## Comparison

| method | speed | accuracy | requires | best for |
|--------|-------|----------|----------|----------|
| `keyframe` | 10-30s | good | - | quick cuts, clean animation |
| `transnetv2` | 1-2 min | best | `amverge[ml]` | complex transitions, fades |
| `edge` | 3-5 min | excellent | `amverge[edge]` | fast motion, difficult encodes |

## Tuning

Keyframe detection has the fewest parameters - scenes split at I-frame
boundaries automatically. For finer control over edge and TransNetV2:

```bash
# TransNetV2: lower threshold = fewer cuts
amverge detect ep.mp4 --method transnetv2 --min-duration 0.5

# Edge: adjust sensitivity
amverge detect ep.mp4 --method edge --edge-threshold 0.15 --edge-radius 0.6
```

| Parameter | Default | Effect |
|-----------|---------|--------|
| `--min-duration` | `0.25` | Merge scenes shorter than N seconds |
| `--edge-threshold` | `0.15` | Lower = fewer cuts (edge method) |
| `--edge-radius` | `0.6` | Keyframe window radius (edge method) |
| `--similarity-threshold` | `0.10` | Lower = stricter similarity merging |
