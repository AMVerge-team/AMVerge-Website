# Exporting

Write your selected (or merged) clips to disk.

## Steps

1. Set the **export directory** in the path field.
2. Choose **Merge Clips** on or off.
3. Click **Export**.

## Output

Clips export as MP4. With merge on you get one file, with merge off you get one
file per selected clip.

That's the full workflow: import, select, merge, export. For internals, see the
[Developer](/docs/architecture) section.
