# Contributing

How to work on AMVerge and these docs.

## Docs site

```bash
npm install
npm run dev
```

Docs live in `src/docs/`. To add a page:

1. Create `src/docs/pages/<name>.mdx`.
2. Register it in `src/docs/registry.ts` under the right group.

Sidebar links, categories, and routes all come from that registry. No other
wiring needed.

## Conventions

- No em dashes in prose or commits. Use commas, colons, or plain hyphens.
- Theme color is always `var(--accent)`, never a hardcoded green.
- Commit tags: `(add)`, `(fix)`, `(update)`.

## Code

The app and detection backend live in separate repos (`AMVerge-team/AMVerge` and
`AMVerge-CLI`). Open issues or PRs there for app behavior.
