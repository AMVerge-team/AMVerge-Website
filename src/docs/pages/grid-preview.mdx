# Grid Preview

Grid Preview plays every clip in the grid in place, so you can skim a whole
episode and spot scenes fast instead of focusing them one at a time.

## Enabling it

Tick **Preview All** in the top toolbar. Clips across the grid start previewing
as you scroll.

## Grid columns

Use the **Grid: N columns** control with the `-` / `+` buttons in the top right
to change how many clips fit per row. Fewer columns = bigger clips; more columns
= a denser overview.

> **Performance:** If too many clips preview at once, Grid Preview gets heavy and
> can slow things down. If that happens, raise the column count until it feels
> smooth again.

## Hover audio

Enable **Audio Playback Hover** in [General Settings](/docs/settings-general) to
hear a clip's audio when you hover it in the grid. Volume is adjustable in the
same place.

## Preview method

In General Settings, **Preview Method** controls how previews are generated:

- **Video files** : preview straight from the source video.
- **WebP files** : generate lightweight animated WebP previews (turns Grid
  Preview on automatically).

Preview playback speed is set by **Grid preview speed** in
[Appearance](/docs/settings-appearance).
