# <span>AMV</span>erge CLI

The AMVerge CLI is a command-line tool and Python library that ports the
desktop app's backend into a standalone package. Install it with `pip` and
run scene detection, export, and more directly from your terminal or from
Python code.

## What it does

- Split videos into individual scenes using keyframe, edge, or ML detection
- Export clips with full codec, bitrate, and format control
- Merge scenes back together after false cuts
- Generate thumbnails and detect similar adjacent scenes
- Discord Rich Presence during long operations
- Fully usable as a Python library (`import amverge`)

## Desktop app vs CLI

The [AMVerge desktop app](/docs/installation) gives you a visual grid browser,
hover preview, and a full GUI. The CLI gives you the same detection and export
engine, but from the terminal. Use whichever fits your workflow.

## Quick start

```bash
pip install amverge
amverge detect episode.mp4
amverge export episode.mp4 --scenes episode_scenes/scenes.json --select 0,2,5-8
```
