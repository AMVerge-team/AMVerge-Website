# Selecting Clips

Scan the grid, preview on hover, and pick the scenes you want.

## Selection controls

- **Click** a clip to preview it in the right pane.
- **Ctrl + Click** to add or remove individual clips.
- **Shift + Click** to select a range.
- **Double Click** for quick select.

## Grid preview

Toggle **Grid Preview** to play every clip in place. Adjust cell size with the
`-` / `+` buttons to fit more scenes on screen.

The selected count shows how many clips are currently picked.
