# Scene Cutting

Split detected scenes into individual clip files.

## Smart Cut

```python
from amverge import cut_scene, cut_all_scenes

# Cut a single scene
cut_scene("episode.mp4", start=10.5, end=25.3, output="scene_01.mp4")

# Cut all detected scenes at once
cut_all_scenes("episode.mp4", scenes, output_dir="./clips")
```

Smart cut uses lossless stream copy for keyframe-aligned scenes, and re-encodes only the head frames (until the next keyframe) for non-aligned boundaries.

## FFmpeg Segmenter

```python
from amverge import run_ffmpeg_segment, collect_scenes

# Segment video at timestamp boundaries
run_ffmpeg_segment("episode.mp4", timestamps, output_dir="./clips")

# Collect resulting clip files
scenes = collect_scenes(output_dir="./clips")
```

Batch splitting via FFmpeg segment muxer. Handles up to 1500 cuts per batch (Windows command-line limit).
