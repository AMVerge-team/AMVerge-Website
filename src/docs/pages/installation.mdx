# Installation

AMVerge is a desktop app. Download a build and run it, no setup required.

## Download

Grab the latest release from the home page or directly from GitHub:

- [Download from the AMVerge site](/#download)
- [GitHub Releases](https://github.com/AMVerge-team/AMVerge/releases)

AMVerge is **free and open source**.

## Platforms

| Platform | Notes |
|---|---|
| **Windows 10/11** | `.exe` installer |
| **macOS** | `.dmg` installer |

## First run

1. Install and open AMVerge.
2. Click **Import Episode** (or drag a video onto the window).
3. Wait for the scene grid to build, then start browsing.

See [Interface Overview](/docs/app-overview) for a tour of the workspace.

## Recommended: HEVC support (Windows)

Many modern encodes use H.265 / HEVC. Installing the official **Microsoft HEVC
Video Extension** makes previews for those videos load much faster. The app's
top bar shows your HEVC status.

More detail: [HEVC Support](/docs/hevc-support).
