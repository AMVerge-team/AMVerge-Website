# Installation

Get AMVerge running locally in a few steps.

## Prerequisites

- Node.js 18+
- A modern browser

## Install

```bash
git clone https://github.com/your-org/amverge.git
cd amverge
npm install
```

## Run dev server

```bash
npm run dev
```

Open `http://localhost:5173` and you are live.

## Build

```bash
npm run build
npm run preview
```

That's it. Back to [Introduction](/docs).
