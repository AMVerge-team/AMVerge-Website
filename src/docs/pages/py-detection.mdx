# Detection Methods

Python API for scene boundary detection.

## Keyframe Detection

```python
from amverge import detect_cuts_by_keyframe

scenes = detect_cuts_by_keyframe("episode.mp4")
```

Fast I-frame based detection. No extra dependencies.

## Edge Detection

```python
from amverge import detect_cuts_by_edge

scenes = detect_cuts_by_edge(
    "episode.mp4",
    threshold=0.15,
    radius=0.6,
)
```

Canny edge maps + cosine similarity. Requires `pip install amverge[edge]`.

## TransNetV2 Detection

```python
from amverge import (
    TRANSNET_AVAILABLE,
    decode_and_detect_scenes,
    run_model_one_pass,
)

if TRANSNET_AVAILABLE:
    scenes = decode_and_detect_scenes("episode.mp4")
```

Deep learning CNN on 48x27 RGB frames. GPU auto-detected, CPU fallback.  
Requires `pip install amverge[ml]`.

## Scene Detector Class

```python
from amverge import SceneDetector

detector = SceneDetector(method="transnetv2")
result = detector.detect("episode.mp4")
```

Configurable class wrapping all three methods with unified output.

## TransNetV2 Config

```python
from amverge import TransNetConfig

config = TransNetConfig(
    frame_width=48,
    frame_height=27,
    device="cuda",  # or "cpu"
)
```
