# General Settings

Open **Settings** from the sidebar, then the **General** tab.

## Options

| Setting | What it does |
|---|---|
| **Scene Detection Method** | How clips are detected. GPU using TransNetV2 is the active method; other options are placeholders. |
| **Preview Method** | Grid previews use **Video files** or generated **WebP files** (WebP turns Grid Preview on). |
| **Audio Playback Hover** | Play a clip's audio when you hover it in the grid. |
| **Playback Volume** | Hover-audio volume, shown as a percentage. |
| **Episodes Storage Path** | Where processed episodes and clips are stored. Changing it moves existing data to the new folder. |
| **Clear Episode Panel** | Remove all episodes and delete their cached files. Does not touch originals. |
| **Factory Reset** | Restore default settings and move storage back to AppData. |
| **Application Version** | The installed AMVerge version. |

## Destructive actions

**Clear Episode Panel** and **Factory Reset** both ask for confirmation first.

- **Clear Episode Panel** deletes cached clip files for every episode. It cannot
  be undone, but your original videos are never touched.
- **Factory Reset** reverts all settings and moves your episode storage back to
  the default AppData location.
