# Diagnostics & IPC

Utility functions for health checks, progress reporting, and system info.

## Diagnostics

```python
from amverge import (
    check_environment,
    get_gpu_info,
    get_versions,
    EnvironmentCheck,
    CheckResult,
)

# Full environment check (Python, FFmpeg, optional deps)
result = check_environment()
print(result.status)  # "ok" or "error"

# GPU and CUDA info
gpu = get_gpu_info()
print(gpu.cuda_available, gpu.device_name)

# All dependency versions
versions = get_versions()
```

## Binaries

```python
from amverge import get_ffmpeg, get_ffprobe, get_binary

ffmpeg_path = get_ffmpeg()      # Resolves from PATH or ./bin/
ffprobe_path = get_ffprobe()    # Same resolution logic
binary = get_binary("ffmpeg")   # Generic resolver
```

## IPC & Progress

```python
from amverge import emit_progress, emit_event, log, check_if_path_exists

emit_progress(50, "Processing...")   # Write "PROGRESS|50|Processing..." to stderr
emit_event("MY_EVENT", payload)      # Write prefixed event to stderr
log("Info message")                   # Plain log to stderr
exists = check_if_path_exists("episode.mp4")  # Bool check
```

Used by the CLI and desktop app sidecar for streaming progress to the UI.

## Discord RPC

```python
from amverge import RPC_AVAILABLE, DiscordRPC

if RPC_AVAILABLE:
    rpc = DiscordRPC()
    rpc.start()
    rpc.update(state="Detecting scenes", details="Episode 01")
    rpc.stop()
```

Requires `pip install amverge[discord]`.
