# Menu & Support

Open **Menu** from the sidebar for app info, logs, and help.

## What's in the Menu

| Page | What it's for |
|---|---|
| **About** | What AMVerge is, the recommended workflow, and tips. |
| **Console** | Logs and errors, useful when reporting a bug. |
| **Patch Notes** | Update logs and what changed between versions. |
| **Credits** | Everyone who helped build the project. |

## Reporting bugs

If something breaks, open the **Console** from the Menu to see the error, then
share it when you file a report. There's a built-in **Bug Report** flow for this.

## Get help

For bug reports, updates, feature requests, or general support, join the
community:

- [AMVerge Discord](https://discord.com/invite/bmXjTgsAaN)
