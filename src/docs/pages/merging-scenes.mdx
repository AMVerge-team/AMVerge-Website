# Merging Scenes

Combine related clips back into one seamless segment.

## Why merge

Because AMVerge splits at [I-Frames](/docs/importing-episodes), some encodes
create extra cuts. Fast motion, flashes, camera shakes, or fight sequences can
land a single scene across several short clips. Merging stitches them back
together, no editor required.

Merging is also useful for building compilations of your favorite moments into a
single exported file.

## How to merge

1. Select the clips you want joined (see [Browsing & Selecting](/docs/selecting-clips)).
2. Enable **Merge Clips** in the export panel.
3. Export. The selected clips become one continuous file.

When merge is enabled, AMVerge asks for a name for the merged file before
exporting. Leave **Merge Clips** off to export each selected clip separately
instead.

## Tip

Select clips in the order they should play, then merge. This keeps your project
panel clean: fewer, larger scenes are faster to browse and import into your
editor.
