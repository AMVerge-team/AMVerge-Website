# <span>AMV</span>erge Docs

Welcome to the AMVerge documentation. AMVerge turns scrubbing through long
episodes into seconds. Every scene shown in a grid so you find the exact
frame fast.

> Built for AMV editors who hate single-frame scrubbing.

## What it does

Instead of dragging across a timeline, AMVerge renders an entire episode as a
clip grid. Scan, click, cut. What took 30 minutes drops under 5.

- **Introduction**: you are here.
- **Installation**: get it running locally.

## Quick taste

```ts
import { mergeClips } from 'amverge'

const out = mergeClips(['01_0109.gif', '01_0110.gif'])
```
